# 职场参谋 Logo 设计说明

## 图形寓意

Logo 由「圆角方形容器 + 对话气泡 + 向上箭头」三层结构组成：

- **圆角方形容器**：紫蓝渐变的方形底座，象征「参谋台」这个承载决策的工作台面，稳重、可信赖。
- **对话气泡**：白色对话框带左下角小尾巴，明确传递「一对一咨询、面对面推演」的产品形态。
- **向上箭头**：嵌入气泡中心的紫色箭头，代表「向上的判断与成长」——每一次对话，都是为下一步决策蓄力。

整体可以一句话口述：**「在对话里，把方向指向上方。」**

## 配色逻辑

主色采用**紫蓝渐变**：

- 起始色 `#4F46E5`（Indigo 600）——理性、冷静、可信赖的咨询专业感
- 结束色 `#A855F7`（Purple 500）——智能、前瞻、AI 时代的科技感

紫蓝渐变是当前 AI/SaaS 产品的主流色域，兼具「专业顾问」的稳重与「智能推演」的现代科技氛围，规避了传统咨询行业常见的深蓝/灰黑带来的沉闷感。

文字色采用更深的同色系渐变 `#3730A3 → #7E22CE`，与图形主色系一致，但对比度更适合中长段阅读。

## 文件清单

| 文件 | 用途 |
| --- | --- |
| `logo-full.svg` | 完整版（图形 + 中文「职场参谋」横排），用于官网 header、邮件签名、对外宣传 |

## 使用建议

- 完整版最小宽度建议 **160px**，避免中文字样模糊；纯图形版最小可缩到 **24px**。
- 主版本（彩色渐变）优先用于品牌正式露出场景；进入第三方平台、需要单色输出时使用 mono 版本。
- 周边留白：四周保留至少与图标圆角等高的安全距离（约 14px @ 64px 尺寸）。
- 中文字体回退顺序：PingFang SC → HarmonyOS Sans SC → Source Han Sans SC → Microsoft YaHei，确保在 macOS / Windows / 鸿蒙生态下均能渲染出现代无衬线效果。
